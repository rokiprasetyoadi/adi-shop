# adi-shop
E-COMMERCE
Mini Project use laravel 10

- Framework
    -- Laravel 10
    -- Boostrap 5
- DB MySql
- PHP Version 8.1.12

How to :
1. Clone Project
2. composer install
3. cp .env example .env
4. Generate app key (php artisan key:generate)
5. Setting env, setting db on env, make db
6. php artisan migrate --seed
7. php artisan storage:link
8. php artisan serve
9. enjoy
